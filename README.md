## Overview

The goal of this project is to uncover systemic barriers to gender equality by analyzing global patterns of violence, reproductive health disparities, and societal inequities, offering actionable insights to inform policies that advance women's equity, dignity, and well-being.

### Directories
- [`data/`](data/): Contains both raw and cleaned datasets used for analysis, including gender inequality indicators, reproductive health metrics, and regional data.
- [`r_script/`](r_script/): Contains all R scripts used for data processing, analysis, and visualization. This includes scripts for data cleaning, tidying, regional analysis (Africa, Asia, China), global comparisons, and figure/table generation.
- [`figure_table/`](figure_table/): Stores all generated outputs, including PNG-format figures and R data files of tables created from the analysis scripts.
- [`memo/`](memo/): Includes .qmd and .html files for the two progress memos submitted during the course, documenting project development and intermediate findings.
- `load_pckgs.R`: Script used to load and manage all required R packages to ensure reproducibility of the analysis.

### Project Reports & Summaries
- `Tran_Chau_final_report.qmd` & `Tran_Chau_final_report.html`
  Full technical report outlining project motivation, data sources, cleaning procedures, analytical approach, key visualizations, and conclusions on global gender inequality.
- `Tran_Chau_exec_summary.qmd` & `Tran_Chau_exec_summary.html`
  Concise executive summary for a non-technical audience, highlighting key findings, regional disparities, and policy-relevant insights.
