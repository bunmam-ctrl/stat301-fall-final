## Basic repo setup for final project

## Overview

The goal of this project is to uncover systemic barriers to gender equality by analyzing global patterns of violence, reproducive health disparities, and societal inequities, offering actionable insights to inform policies that advance women's equity, dignity, and well-beging

## Basic Repo Set up 

`proggress memo`: folders contains all the html and qmd of two progress memos
`R script africa + asia`: folder contains all R script of figures and tables for `developing nations in Africa and Asia`
`R script Asia`: folder contains R script of figures and tables for `China`
`R script data`: folder contains R script to produce table and graph summarizing the quality of the datasets
`R script tidy`: folder contains R script to read and tidy datasets
`R script world`: folder contains R script to generate global maps abort number of abortion cases and GII
`data`: folder contains raw data and tidied data
`figure_asia_africq`: png formats of figures and R data file of tables created by `R script for africa + asia`
`figure_china`: png formats of figures and R data file of tables created by `R script china`
`figure_data`: png formats of figures and R data file of tables created by `R script data`
`figure_world`: png formats of map generated by `R script world`
