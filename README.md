## Overview

The goal of this project is to uncover systemic barriers to gender equality by analyzing global patterns of violence, reproducive health disparities, and societal inequities, offering actionable insights to inform policies that advance women's equity, dignity, and well-beging

## Basic Repo Set up 

`progress memo`: Folder contains all the HTML and QMD files for two progress memos.

`R script africa + asia`: Folder contains all R scripts used to create figures and tables for **developing nations in Africa and Asia**.

`R script Asia`: Folder contains R scripts for generating figures and tables specifically for **China**.

`R script data`: Folder contains R scripts for producing tables and graphs summarizing the quality of the datasets.

`R script tidy`:  Folder contains R scripts designed to read and tidy the datasets.

`R script world`: Folder contains R scripts for generating global maps of **abortion numbers and GII data**.

`data`: Folder includes both raw and tidied data files.

`figure_asia_africq`: Folder contains PNG-format figures and R data files of tables created by the `R scripts africa + asia`.

`figure_china`: Folder includes PNG-format figures and R data files of tables created by `R script china`.

`figure_data`: Folder holds PNG-format figures and R data files of tables created by `R script data`.

`figure_world`: Folder contains PNG-format maps created by `R script world`
