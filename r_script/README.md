Storing all R scripts for analyzing, including making figures and tables used in the final reports.

### Directories

- [`africa_asia/`](africa_asia/): Contains R scripts for analyzing gender inequality patterns across countries in Africa and Asia, including regional comparisons and visualizations.
- [`china/`](china/): Includes scripts focused specifically on China, generating country-level figures and tables for more detailed analysis.
- [`tidy/`](tidy/): Contains scripts for data cleaning and preprocessing, including reading raw datasets, tidying variables, and preparing analysis-ready data.
- [`world_vis/`](world_vis/): Stores scripts and outputs for global visualizations, including maps and comparative figures of gender inequality indicators worldwide.
